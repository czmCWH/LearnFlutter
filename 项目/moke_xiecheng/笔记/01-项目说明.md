# 一、项目基本使用

```shell
# 终端命令创建一个 Flutter 项目
$ flutter create flutter_ui

# 确定永远不需要移动端，即可以手动删除
$ rm -rf android ios macos windows linux

# 拉取所有项目依赖
$ flutter pub get

# 升级 carousel_slider 插件
$ flutter pub add carousel_slider

# 运行项目到浏览器
$ flutter run -d chrome

# 清理项目所有依赖
$ flutter clean
```

# 三、重要知识点

## 1、http 请求中文乱码

```dart
// 解决返回数据中文乱码
Utf8Decoder utf8decoder = const Utf8Decoder();  
String bodyString = utf8decoder.convert(response.bodyBytes);
debugPrint('--- 请求结果 = $bodyString');
```

## 2、FutureBuilder 在 main 中初始化数据

```dart
FutureBuilder(   
  future: _doInit(), 
  builder: (BuildContext context, AsyncSnapshot<dynamic> snapshot) {
  }
}
```

## 3、UI处理
- 自定义 MaterialButton ；
- Positioned.fill 填充 Stack 内容；
- Scaffold.resizeToAvoidBottomInset: false, 防止键盘弹起影响布局；
- 用 url_launcher 插件在浏览器中打开网页地址
