# 一、项目基本使用

```shell
# 终端命令创建一个 Flutter 项目
$ flutter create flutter_ui

# 确定永远不需要移动端，即可以手动删除
$ rm -rf android ios macos windows linux

# 拉取所有项目依赖
$ flutter pub get

# 升级 carousel_slider 插件
$ flutter pub add carousel_slider

# 运行项目到浏览器
$ flutter run -d chrome

# 清理项目所有依赖
$ flutter clean
```
