# 一、项目主要功能
  - Navigator 命名路由；
  - UI 封装： 
      carousel_slider 轮播图插件；
      showSnackBar 底部 Toast；
      Dialog 自定义 Loading；
      AlertDialog 自定义弹窗；
  - MediaQuery.of(context) API 获取屏幕Size、安全距离；
  - Dio 插件封装网络请求；
  - shared_preferences 插件本地数据存储；
  - Getx 全局状态共享；

运行项目：
```shell
# 拉取项目依赖
$ flutter pub get
# 运行项目到浏览器
$ flutter run -d chrome

# 清理所有依赖，使得工程占用最小
$ flutter clean
```
