# 三、更新 Flutter
<https://docs.flutter.cn/release/upgrade>

## Flutter 渠道

Flutter 有四个发布渠道：
* `stable`(稳定版), 适用于生产环境。
* `beta`(测试版), 每个月都会选取上个月的最佳开发版本，然后合并到beta版本进行推广测试。
* `master`(主干)，仓库的最新代码分支，供贡献者使用。

```
// 查看当前 Flutter 通道
$ flutter channel

// 切换至 dev 
$ flutter channel dev
```
## 更新 Flutter

```
// 更新 当前 Flutter 通道上可用的、最新版本的Flutter SDK、Dart
$ flutter upgrade 

// 把 Flutter 更新到指定的版本号
$ flutter upgrade v2.2.3
```
