* 收集好用的Flutter插件以便更效率的开发：<https://github.com/jahnli/awesome-flutter-plugins>


Flutter Favorite 项目，<https://docs.flutter.cn/packages-and-plugins/favorites>



```shell
# Dio 网络请求
$ flutter pub add dio

# preference 本地存储
$ flutter pub add shared_preferences

# 轮播图
$ flutter pub add carousel_slider

# 加载 h5 插件
$ flutter pub add webview_flutter 

# 插件用于处理文件存储目录
$ flutter pub add path_provider 

# 跳转到浏览器，访问网页URL
$ flutter pub add url_launcher
```
